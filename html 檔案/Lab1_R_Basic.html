R Basic¶

變數¶

  • R 在設定變數值時是利用「<-」並不是程式語言中常見的「=」，在 R 官方文件都強調不該使用「=」，因為在某些狀況是會失效的。另外在 R 變數命名上，大小寫是有區別的，所以 x 與 X 其實是不同的變數。

c(...) 類似的函數 x:y、seq 與 rep¶

  • seq(s, e, by)：產生一個等差級數的向量。
    • s 是初始值
    • e 是結束值
    • by 是遞增值，預設是 1
  • rep(x, times, each)：產生一個重覆循環的向量。
    • x 是需重覆循環的數值
    • times 是重覆循環次數
    • each 是 x 內元素重覆的次數
In [39]:
seq(5,10,2)
  1. 5

迴圈¶

  • for
In [97]:
y <- 0
for (x in c(1:10)) 
    y <- x + y 
y
55
  • while
In [100]:
x <- 1
y <- 0
while (x <= 10) { # while 只要符合判斷式，就會一直重複執行括號內程式碼，直到不符合為止。
   y <- x + y
  x <- x + 1 # 這行很重要，如果沒有這行，程式碼會一直執行不會停止，因為判斷式是 x 小於等於 10，x 初始值是 0，如果不對 x 做些動作，x 會一直小於等於 10，所以這邊加 1，是希望執行到 x = 11 時，迴圈就會停止。
 }
y
55
