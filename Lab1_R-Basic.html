c(...) 類似的函數 x:y、seq 與 rep¶

  • seq(s, e, by)：產生一個等差級數的向量。
    • s 是初始值
    • e 是結束值
    • by 是遞增值，預設是 1
  • rep(x, times, each)：產生一個重覆循環的向量。
    • x 是需重覆循環的數值
    • times 是重覆循環次數
    • each 是 x 內元素重覆的次數

x <- 1
y <- 0
while (x <= 10) { # while 只要符合判斷式，就會一直重複執行括號內程式碼，直到不符合為止。
   y <- x + y
  x <- x + 1 # 這行很重要，如果沒有這行，程式碼會一直執行不會停止，因為判斷式是 x 小於等於 10，x 初始值是 0，如果不對 x 做些動作，x 會一直小於等於 10，所以這邊加 1，是希望執行到 x = 11 時，迴圈就會停止。
 }

5. 為了建構「樹」的識別系統，一家公司決定在網路上收集一百萬張圖片。¶

然後，公司要求10名員工，每人查看100張圖片並標註每張圖片是否包含樹。¶

再把圖片和標註的紀錄送到「學習算法」以構建系統。 該算法需要解決哪種學習問題？¶

  1. 監督式學習(Supervised Learning)
  2. 非監督式學習(Unsupervised Learning)
  3. 半監督式學習(Semi-supervised Learning )
  4. 強化學習(Reinforcement Learning)

Ans:(3)，因為有1000張照片是有被標註的，而另外的999000張照片是沒有被標註的，所以有包含標註即沒有標註的資料適用於半監督式學習。¶
